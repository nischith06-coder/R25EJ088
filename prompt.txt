## Prompt 9

### User Input

```text
Rectangle is going infinite.
```

### Output

Identified issue:

```c
#define WIDTH 600
#define HEIGHT 250
```

Large canvas size produced excessive console output.

Recommended:

```c
#define WIDTH 60
#define HEIGHT 25
```

for console display.
