## 8. Canvas Size Issue

### Prompt

Rectangle is going infinite.

### Output

Identified issue:

```c
WIDTH = 600
HEIGHT = 250
```

causing massive console output.

Recommended:

```c
WIDTH = 60
HEIGHT = 25
```

for terminal rendering.
